# Pwmenu  
**Идеальный менеджер паролей для Pwnagotchi**

Мощный плагин для управления захваченными паролями и хендшейками прямо на вашем Pwnagotchi.  
Красивый тёмный интерфейс в стиле iOS, интеграция с WPA-Sec, система достижений и удобные инструменты для работы с `.pcap` файлами.

![image](https://github.com/newfpv/pwmenu/blob/main/image.webp)

## ✨ Возможности
- **Красивый веб-интерфейс:** тёмная iOS-тема, вкладки, анимации, простая навигация.
- **Группировка сетей:** все хендшейки для одной сети (ESSID) собираются в одну группу.
- **Менеджер паролей:**
  - просмотр всех найденных паролей;
  - ручное добавление, редактирование и удаление записей;
  - импорт паролей из `.json` и `.csv` (например, экспортов OnlineHashCrack).
- **Работа с файлами:**
  - скачивание `.pcap` файлов;
  - автоматическая конвертация в `.hc22000` (формат Hashcat) прямо на устройстве;
  - удаление отдельных файлов;
  - массовая загрузка всех хендшейков в одном ZIP-архиве.
- **Интеграция с WPA-Sec:** загрузка хендшейков на wpa-sec.stanev.org в один клик.
- **Инструменты обслуживания:**
  - обнаружение и удаление битых или пустых дампов;
  - полная очистка данных (Nuke All).
- **Система достижений:** XP, уровни и бейджи.
- **Синхронизация времени:** кнопка для быстрой синхронизации времени через Google.

## 🚀 Установка
```bash
ssh pi@10.0.0.2
cd /usr/local/share/pwnagotchi/custom-plugins/
sudo wget https://raw.githubusercontent.com/newfpv/pwmenu/main/A_pwmenu.py
sudo nano /etc/pwnagotchi/config.toml
```
Добавьте конфигурацию плагина:
```toml
main.plugins.A_pwmenu.enabled = true
main.plugins.A_pwmenu.wpa_sec_key = "YOUR_WPA_SEC_KEY"
main.plugins.A_pwmenu.timezone = 3
```
Перезапустите Pwnagotchi:
```bash
sudo systemctl restart pwnagotchi
```

## 📖 Руководство пользователя
### Вкладка «Cracked»
Показывает все найденные пароли.
- **Бейдж источника** показывает, откуда пришёл пароль (WPA-Sec, OHC, Manual, QuickDic).
- **✏️ Edit** — изменить пароль.
- **✖ Delete** — удалить запись.

### Вкладка «Handshakes»
Список захваченных сетей.
- нажмите на сеть, чтобы развернуть все `.pcap` файлы для этого ESSID;
- зелёное название сети означает, что пароль уже найден.

Действия внутри группы:
- **WPA:** загрузить файл на wpa-sec;
- **22000:** скачать версию `.hc22000`;
- **PCAP:** скачать оригинальный дамп;
- **Delete:** удалить файл;
- **Add:** добавить пароль вручную.

### Вкладка «Other»
- **Достижения:** уровень, XP и бейджи.
- **Статистика:** общая статистика.
- **Экспорт:** список ESSID:PASSWORD или ZIP с файлами.
- **Импорт:** загрузка файлов паролей в формате `.json`/`.csv`.
- **Danger Zone:** удаление битых файлов или полная очистка данных.

## 🛠 Устранение неполадок
### Conversion Failed
`.pcap` пустой или не содержит валидный PMKID/EAPOL. Используйте кнопку "Clean Broken Files".

### Нет кнопки WPA-Sec
Добавьте `main.plugins.A_pwmenu.wpa_sec_key` в `config.toml`.

### Неправильное время
Отрегулируйте `main.plugins.A_pwmenu.timezone`.
